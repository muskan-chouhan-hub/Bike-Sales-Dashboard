# Bike Sales Dashboard

## Project Overview
This project analyzes bike sales data to understand customer behavior, sales trends, and revenue performance across regions, age groups, and income levels.

## Tools Used
- Power BI
- Excel

## Key Features
- Sales by Region
- Customer Age Group Analysis
- Income vs Purchase Behavior
- Gender-wise Sales Distribution
- Monthly Sales Trends

## Key Insights
- Identified top-performing regions
- Found key customer segments driving sales
- Analyzed income impact on purchase decisions
